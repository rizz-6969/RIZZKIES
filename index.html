1. Apa itu Web 3?

Web 3 adalah generasi internet berbasis blockchain yang mendukung desentralisasi dan kepemilikan data oleh pengguna.

2. Apa keunggulan Web 3 dibanding Web 2?

Pengguna Web 3 memiliki kontrol penuh atas data mereka dan tidak tergantung pada platform terpusat.

3. Apa itu smart contract?

Smart contract adalah kode yang berjalan otomatis di blockchain untuk mengeksekusi kesepakatan tanpa perantara.

4. Apa peran token di Web 3?

Token digunakan untuk insentif, akses layanan, dan tata kelola dalam ekosistem Web 3.

5. Apa itu DAO?

DAO (Decentralized Autonomous Organization) adalah organisasi yang dikelola oleh kode dan komunitas melalui voting berbasis token.
