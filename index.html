FAQ Web 3

1. Apa itu Web 3?
Web 3 adalah generasi internet berikutnya yang terdesentralisasi dan memberi pengguna kendali atas data mereka sendiri.
2. Apa perbedaan Web 2 dan Web 3?
Web 2 bersifat terpusat dan dikendalikan oleh perusahaan besar, sementara Web 3 dikontrol oleh komunitas dan lebih aman.
3. Apa itu blockchain?
Blockchain adalah sistem pencatatan digital yang tersebar dan aman, menjadi dasar dari Web 3.
