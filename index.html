35 Pertanyaan & Jawaban Seputar Web 3

1. Apa itu Web 3?

Web 3 adalah generasi internet berikutnya yang bersifat terdesentralisasi, memberi pengguna kendali atas data dan identitas mereka.

2. Apa perbedaan Web 2 dan Web 3?

Web 2 dikontrol oleh perusahaan besar, sedangkan Web 3 dikendalikan oleh komunitas melalui teknologi blockchain.

3. Mengapa Web 3 penting?

Web 3 menjanjikan transparansi, privasi, dan kepemilikan data bagi pengguna.

4. Teknologi utama Web 3?

Blockchain, smart contract, IPFS, dan kriptografi.

5. Siapa yang mengembangkan Web 3?

Dikembangkan oleh komunitas terbuka, startup blockchain, dan organisasi seperti Ethereum Foundation.

6. Apa itu dApps?

Decentralized Applications yang berjalan di atas blockchain tanpa kontrol pihak pusat.

7. Fungsi wallet Web 3?

Menyimpan aset kripto dan mengakses dApps secara aman.

8. Apa itu MetaMask?

Dompet digital berbasis browser untuk mengelola aset dan berinteraksi dengan Web 3.

9. Apa itu gas fee?

Biaya transaksi untuk memproses aktivitas di blockchain.

10. Blockchain populer untuk Web 3?

Ethereum, Solana, Polkadot, dan Avalanche.

11. Apa itu NFT?

Non-Fungible Token, aset digital unik di blockchain, seperti karya seni dan koleksi.

12. Bagaimana staking bekerja?

Menyimpan token untuk membantu validasi transaksi dan mendapatkan imbalan.

13. Apa itu Layer 2?

Solusi di atas blockchain utama untuk meningkatkan kecepatan dan biaya rendah, seperti Optimism dan zk-rollup.

14. Web 3 & DeFi?

DeFi adalah bagian dari Web 3 yang memungkinkan layanan keuangan tanpa bank atau lembaga pusat.

15. Apa itu yield farming?

Strategi untuk mendapatkan imbal hasil dengan menyediakan likuiditas di DeFi.

16. Governance token?

Token yang memberi hak suara dalam proyek blockchain.

17. Apa itu IPFS?

InterPlanetary File System, sistem penyimpanan file terdistribusi yang tidak bergantung pada server pusat.

18. Apakah Web 3 aman?

Secara teori lebih aman, tapi tetap ada risiko dari bug dalam smart contract dan penipuan.

19. Tantangan utama Web 3?

Skalabilitas, adopsi massal, UX, regulasi, dan interoperabilitas.

20. Apa itu cross-chain?

Komunikasi dan pertukaran aset antar blockchain berbeda.

21. Fungsi bridge?

Memindahkan aset antar jaringan blockchain.

22. Apa itu oracle?

Penghubung antara blockchain dan data dunia nyata.

23. Apa itu zk-rollup?

Teknologi Layer 2 yang merangkum banyak transaksi menjadi satu bukti kriptografi.

24. Web 3 & privasi?

Pengguna punya kendali atas data pribadi dan bisa anonim.

25. Desentralisasi?

Distribusi kontrol dan otoritas kepada banyak pihak, bukan entitas tunggal.

26. Apa itu tokenomics?

Ekonomi di balik sebuah token: distribusi, suplai, dan insentif.

27. Web3 Foundation?

Organisasi nirlaba yang mendukung pengembangan teknologi Web 3, termasuk Polkadot.

28. Konsensus blockchain?

Cara jaringan menyepakati transaksi yang valid, seperti Proof of Work dan Proof of Stake.

29. Proof of Stake?

Metode validasi transaksi berdasarkan jumlah token yang dikunci oleh validator.

30. Identitas Web 3?

Identitas digital terdesentralisasi yang dikendalikan oleh pengguna.

31. DID (Decentralized ID)?

Teknologi identitas yang tidak bergantung pada pihak ketiga.

32. Cara belajar Web 3?

Gunakan platform seperti CryptoZombies, Buildspace, Coursera, dan ikut komunitas developer Web 3.

33. Apa itu testnet?

Blockchain uji coba untuk eksperimen sebelum ke jaringan utama.

34. Apakah Web 3 legal?

Bervariasi tergantung negara; beberapa mengatur aset digital, lainnya belum.

35. Apa masa depan Web 3?

Web 3 berpotensi menjadi fondasi internet baru yang lebih adil, terbuka, dan dikendalikan oleh komunitas.
