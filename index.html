FAQ Web 3 (35 Pertanyaan)

1. Apa itu Web 3?
Web 3 adalah generasi internet berikutnya yang terdesentralisasi, memberi kontrol data kembali ke pengguna.
2. Apa perbedaan Web 2 dan Web 3?
Web 2 bersifat terpusat, sementara Web 3 menggunakan blockchain untuk mendesentralisasi kendali.
3. Mengapa Web 3 penting?
Karena menawarkan transparansi, privasi, dan kepemilikan digital kepada pengguna.
4. Apa itu blockchain?
Blockchain adalah buku besar digital terdesentralisasi yang menyimpan transaksi secara aman.
5. Apa itu smart contract?
Smart contract adalah kode yang dieksekusi otomatis bila kondisi terpenuhi, berjalan di blockchain.
6. Apa itu dApp?
dApp adalah aplikasi terdesentralisasi yang berjalan di jaringan blockchain tanpa entitas pusat.
7. Apa itu token?
Token adalah aset digital di blockchain, dapat bersifat fungible maupun non-fungible.
8. Apa itu NFT?
NFT adalah token unik yang mewakili kepemilikan digital atas karya karya seni, musik, dll.
9. Apa itu wallet Web 3?
Wallet adalah dompet digital untuk menyimpan aset, berinteraksi dengan dApp, dan mengelola private key.
10. Apa itu MetaMask?
MetaMask adalah wallet populer berbasis browser untuk mengakses Ethereum dan dApp Web 3.
11. Apa itu gas fee?
Gas fee adalah biaya transaksi untuk digunakan dalam jaringan blockchain seperti Ethereum.
12. Apa itu staking?
Staking adalah mengunci token di jaringan blockchain untuk memvalidasi transaksi dan mendapat imbalan.
13. Apa itu Layer 2?
Layer 2 adalah solusi tambahan untuk meningkatkan kecepatan dan menurunkan biaya transaksi di Layer 1.
14. Apa itu DeFi?
DeFi adalah layanan keuangan tanpa perantara menggunakan smart contract di blockchain.
15. Apa itu yield farming?
Yield farming adalah strategi mendapatkan imbal hasil lewat likuiditas DeFi.
16. Apa itu governance token?
Token yang memberi hak suara dalam keputusan pengembangan proyek Web 3.
17. Apa itu IPFS?
IPFS adalah sistem penyimpanan file terdistribusi untuk menjaga desentralisasi data.
18. Apakah Web 3 aman?
Dirancang lebih aman lewat kriptografi, namun masih rentan jika smart contract tidak audit.
19. Apa tantangan utama Web 3?
Skalabilitas, UX, interoperabilitas, serta regulasi masih menjadi tantangan.
20. Apa itu cross‑chain?
Cross‑chain memungkinkan interoperabilitas antar blockchain berbeda.
21. Apa itu bridge?
Bridge memungkinkan pemindahan aset antar blockchain.
22. Apa itu oracle?
Oracle adalah perantara data off‑chain ke smart contract.
23. Apa itu zk‑rollups?
zk‑rollups adalah teknologi Layer 2 untuk menggabungkan banyak transaksi ke dalam satu bukti kriptografi.
24. Bagaimana Web 3 mendukung privasi?
Dengan memberi kontrol data ke pengguna dan mendukung identitas anonim sesuai kebutuhan.
25. Apa itu desentralisasi?
Desentralisasi berarti kontrol jaringan tidak dimonopoli satu pihak, tetapi tersebar ke banyak node.
26. Apa itu tokenomics?
Tokenomics adalah studi ekonomi token: suplai, distribusi, insentif, dan nilai pasar.
27. Apa itu Web3 Foundation?
Organisasi nirlaba yang mendorong pengembangan teknologi Web3, termasuk Polkadot.
28. Apa itu mekanisme konsensus?
Metode validasi transaksi di blockchain, seperti Proof of Work atau Proof of Stake.
29. Apa itu Proof of Stake?
Proof of Stake adalah sistem dimana validator memegang token untuk memverifikasi transaksi.
30. Apa itu decentralized identity?
Identitas digital yang pengguna kendalikan, tanpa pihak ketiga.
31. Apa itu DID?
Decentralized Identifier, sistem identitas digital yang aman dan terdesentralisasi.
32. Bagaimana cara belajar Web 3?
Lewat dokumentasi, tutorial, platform edukasi seperti CryptoZombies atau Buildspace.
33. Apa itu testnet?
Jaringan uji coba untuk mencoba aplikasi sebelum dijalankan di mainnet.
34. Apakah Web 3 legal?
Bergantung pada regulasi tiap negara—beberapa mengatur crypto dan NFT, lainnya belum.
35. Masa depan Web 3?
Potensinya tinggi untuk membuka internet yang lebih adil, inklusif, dan komunitas‑driven.
