1. Apa itu Web 3?

Web 3 adalah evolusi internet yang berbasis teknologi blockchain, memungkinkan kontrol data oleh pengguna.

2. Apa perbedaan Web 2 dan Web 3?

Web 2 dikendalikan oleh platform besar, sedangkan Web 3 lebih terdesentralisasi dan memberi pengguna kendali atas data mereka.

3. Apa itu blockchain dalam konteks Web 3?

Blockchain adalah teknologi inti Web 3 yang menyimpan data secara aman, transparan, dan terdistribusi.
